# Ruta Frontend Fullstack: De los Fundamentos a la Reactividad con Vue.js

¡Bienvenidos a la guía definitiva de desarrollo web modular! Este repositorio contiene el material didáctico estructurado, los mini-proyectos y los desafíos prácticos diseñados para transformarte en un desarrollador Frontend ágil, eficiente y con mentalidad de Arquitecto de Software.

Olvídate de tutoriales infinitos y explicaciones redundantes. Aquí vamos directo al grano, priorizando la calidad del código, el rendimiento y la escalabilidad futura.

---

## 🎯 Promesa del Curso

Al finalizar esta ruta de aprendizaje, no solo sabrás escribir código en HTML, CSS, JavaScript, Bootstrap y Vue.js; entenderás **por qué** y **cuándo** utilizar cada herramienta, dominando la transición entre el desarrollo imperativo tradicional y las arquitecturas reactivas modernas.

---

## 🛠️ Pilares Metodológicos (Nuestras Reglas de Oro)

Para garantizar un aprendizaje limpio y profesional, todo el material de este curso se rige bajo cinco principios estrictos:

* **Código Minimalista:** Menos es más. Si un bloque lógico funciona de manera óptima en 5 líneas, no usaremos 10. Priorizamos el bajo consumo de recursos del navegador.
* **Clean Code Autodocumentado:** Código modular y semántico. Las variables y funciones están tan bien estructuradas que el código se explica solo, minimizando los comentarios innecesarios.
* **Enseñanza Comparativa:** Cada concepto avanzado en Vue.js se analiza en paralelo con su equivalente en **Vanilla JavaScript**, facilitando una transición mental sólida sin "magia negra".
* **Planificación Proactiva:** Antes de tirar una sola línea de código en los proyectos, analizamos la estructura arquitectónica para asegurar la mantenibilidad a largo plazo.
* **Progresión Estricta:** Un paso a la vez. Ningún nivel o proyecto utilizará etiquetas, APIs o funciones que no hayan sido explicadas explícitamente en módulos anteriores.

| Nivel | Enfoque Tecnológico | Competencia Clave | Proyecto Práctico |
| :--- | :--- | :--- | :--- |
| **Nivel 1** | HTML5, CSS3 Básica & Vanilla JS | Estructura Semántica y Manipulación Básica del DOM. | **Monitor de Estado de Sistema** |

| **Nivel 6** | Integración Total Full Frontend | Arquitectura de Componentes, Consumo de APIs y Estilos con Bootstrap. | **E-Commerce Completo con Carrito** |

## 🚀 Cómo Usar Este Repositorio

Cada nivel cuenta con su propia carpeta independiente estructurada de la siguiente manera:

```text
📂 nivel-X/
├── 📄 README.md          # Conceptos clave, Buenas prácticas y Trivia
├── 📂 ejemplos/          # Snippets de código aislados y minimalistas
└── 📂 mini-proyecto/     # Código fuente del software desarrollado en clase
```

### Instrucciones de Inicio Rápido
1. **Clona este repositorio:**
   ```bash
   git clone [https://github.com/tu-usuario/curso-frontend-vue.git](https://github.com/tu-usuario/curso-frontend-vue.git)
   ```
2. **Navega al nivel deseado:**
   ```bash
   cd curso-frontend-vue/nivel-1
   ```
3. **Abre el archivo `index.html` en tu navegador** preferido o utiliza la extensión *Live Server* en tu editor de código para ver los cambios en tiempo real.
